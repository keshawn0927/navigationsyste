# FastRoute Delivery Navigation System

This is a Python delivery route app that helps a delivery driver enter route information and calculate a route using the Google Maps API.

## Features

- Enter pickup location
- Enter delivery destination
- Enter client name
- Enter driver name
- Calculate route
- Show route summary
- Display total distance and estimated travel time
- Open a route map in the browser
- Show company information

## Technologies Used

- Python
- Tkinter
- Google Maps API
- Folium
- python-dotenv
